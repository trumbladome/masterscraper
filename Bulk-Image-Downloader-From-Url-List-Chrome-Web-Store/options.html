1. Introduction

This tool revolutionizes how you collect and organize web images, offering enterprise-grade downloading capabilities in a simple Chrome extension.

  • Scheduled downloads - Set precise future download times down to the minute
  • Queued downloads - Process tasks sequentially for reliable large batches
  • Dynamic filenames - Construct custom names using URL segments, sequences, and metadata
  • Smart filtering - Download only if URLs meet specific "Contains/Not Contains" conditions
  • CSV import/export - Manage thousands of URLs via spreadsheet integration
  • License tiers - Free (5 URLs/task) and Premium (unlimited) versions
  • ZIP packaging - Automatically compress downloads into organized archives
  • Duplicate prevention - Automatic URL deduplication within tasks

Perfect for: E-commerce product catalogs ‚Ä¢ Image archive migration ‚Ä¢ Content aggregation ‚Ä¢ Research datasets

5. Task Management

Efficiently organize and manage your download tasks with these powerful controls.

Lock/Unlock Drag

The Lock/Unlock Drag feature allows users to control whether tasks can be rearranged by dragging.

Draft Save

The Draft Save feature allows users to save tasks without triggering downloads:

Creating a New Task

  1. Click "Create Task" (below the default task)
  2. Configure all settings:
    • URLs - Paste one URL per line (max 5 in free version)
    • File Types - Select formats or choose "All Types"
    • Folder Name - Optional subfolder for organization
    • Download Time - Future scheduling (uses your local timezone)
    • ZIP Download - Toggle to package files automatically
    • Filename Constructor - Enable for custom naming patterns
    • Download If - Set conditional filters if needed
    • Convert - Download a different format other that that of your original images e.g PNG to WebP
    • Quality - Default quality is 100%. Downgrade the quality slightly (no significant quality changes) by adjusting it to, say 80% to reduce the image size.
  3. Click to Save. Use (Save Options to Trigger the Schedule or Queued downloads OR click the Draft Save to save tasks without triggering downloads.

Editing an Existing Task

  1. Locate the task you want to modify
  2. Change any field:
    • URL list (add/remove/edit)
    • File type selections
    • Download timing
    • Folder destination
  3. Click "Save Options" (floating button on right edge or below the default task or on the top left for the Draft Save) to apply changes
  4. For immediate changes to scheduled tasks:
    • Cancel and recreate the task

Duplicating a Task

  1. Find the task you want to duplicate
  2. Click the "Duplicate" button (bottom-right of task)
  3. New task appears with:
    • All original settings
    • New unique ID
    • "Pending" status
  4. Note: Duplicated tasks maintain their own sequence numbers. This means a duplicated task is technically a new unique task with the settings of the parent task. Duplicate tasks save automatically on clicking the Duplicate button.

Deleting a Single Task

  1. Locate the task to remove
  2. Click the red "X" button (bottom-right corner)
  3. Task disappears immediately without confirmation. This is to prevent multiple confirmations when you are deleting many specific tasks.
  4. For recovery:
    • Recreate manually

Deleting All Tasks

  1. Click the floating "Remove All Tasks" button (right screen edge)
  2. Confirm deletion in the popup dialog
  3. All tasks will be permanently removed
  4. Alternative options:
    • Delete tasks individually
Important Notes:
  • Deleted tasks cannot be recovered
  • Active downloads will complete before deletion takes effect
  • The "Save Options" button saves all task changes simultaneously
Workflow Tip: Use CSV exports to:
  • Backup your task configurations
  • Transfer tasks between browsers/devices

8. Download Options

Flexible download methods to suit different needs and connection types.

Scheduled Downloads

  1. Set a precise future time using the datetime picker
  2. The extension's background process checks every 30 seconds for pending tasks
  3. Features:
    • Runs exactly at specified time (nearest 30-second interval)
    • Respects system sleep/wake cycles

Queued Downloads

  1. Toggle the "Queued Download" switch at the top
  2. Tasks process sequentially:
    • One task completes fully before next begins
    • Files within a task download consecutively
    • Progress bar shows queue position
  3. Automatic features:
    • Pauses on connection loss
    • Resumes from last successful download
    • Skips failed items after 3 attempts

ZIP Download

  1. Enable the "Zip" toggle in task options
  2. Configuration options:
    • ZIP filename matches folder name
    • Maintains original file structure
    • Standard ZIP compression (no password)
  3. System handles:
    • Automatic retries if ZIP creation fails
    • Validation of archive integrity
    • Cleanup of temporary files

Automatic Retry System

The extension automatically retries failed downloads with:

  • 3 attempts per file
  • 30-second intervals between retries
  • Exponential backoff on server errors
Pro Tips:
  • Use Queued Mode for slow connections or server-side rate limits
  • Combine Scheduled and ZIP for overnight backups
  • Monitor progress via the browser's native download manager
Note:
  • Scheduled tasks require Chrome to be running at the trigger time
  • ZIP creation may fail for very large collections (>2GB)
  • Queued mode disables parallel downloads across all tasks

9. Filename Constructor

The filename constructor provides powerful customization for organizing downloaded files exactly how you need them.

Customizing Filenames

  1. Enable the "Filename Constructor" checkbox
  2. Build your filename using these components:
    • Filename - Original name from URL
    • Sequence - Auto-incrementing number with zero-padding
    • Timestamp - Current download time in milliseconds
    • URL Fragments - Path segments (e.g., "products" from example.com/products/)
    • Website - Domain name (e.g., "example-com")
    • Random String - 8-character unique ID
    • Custom text - Any text you enter manually
    • URL Segments - Automatically detected when pasting structured URLs
  3. Drag and drop components to arrange them
  4. The system automatically handles:
    • File extensions (always positioned last)
    • Duplicate filename prevention
    • Special character sanitization

Advanced Sequence Numbering

  1. Set your starting number in the "Start from" field:
    • Simple numbers: 1, 42
    • Padded numbers: 001, 0050
  2. The system will:
    • Maintain your padding style (e.g., 0050 ‚Üí 0051)
    • Increment automatically for each file
    • Apply sequence numbers exactly where placed in your pattern

Smart URL Processing

When pasting URLs with consistent structures, the system automatically detects available segments:

Example URL: https://example.com/images/2023/products/photo123.jpg

Detected Segments:

  • Segment 1: example
  • Segment 2: com
  • Segment 3: images
  • Segment 4: 2023
  • Segment 5: products

Example filename patterns:

  • photo_001.jpg (basic sequence)
  • example-com_2023-products_0001.jpg (website + year + category)
  • product-[sequence]_[timestamp].jpg (custom pattern)
  • downloaded_[Segment4]_[Segment5]_[random].jpg (URL segments + random ID)

Note: The ".extension" component is automatically positioned and cannot be removed.

Pro Tip: Combine URL segments with sequence numbers for perfectly organized downloads.
Example: [Segment4]/[Segment5]_[sequence].[extension] creates "2023/products_0001.jpg"

10. Download Conditions (Download IF)

Advanced URL filtering that gives you precise control over which files get downloaded.

Setting Up Conditions

  1. Enable the "DOWNLOAD IF URL" checkbox
  2. Create rules using:
    • Contains - URL must include this text
      Example: product matches "product-image.jpg"
    • Not Contains - URL must exclude this text
      Example: thumbnail excludes "product-thumbnail.jpg"
  3. Click "Add" to save each condition
  4. The system automatically:
    • Prevents duplicate conditions
    • Shows active conditions in a list
    • Updates the AND/OR logic options when needed

Advanced Logic Options

  1. When using both Contains and Not Contains:
    • AND - Both conditions must be true
      Example: Must contain "summer" AND not contain "sale"
    • OR - Either condition must be true
      Example: Must contain "blue" OR not contain "red"
  2. Logic options automatically appear when needed
  3. Single condition types work without logic selection

Practical Examples

Common use cases:

  • Contains: "high-res" - Only downloads high resolution versions
  • Not Contains: "/thumbnails/" - Excludes thumbnail directory
  • Contains: "product" AND Not Contains: "discontinued" - Current products only
  • Contains: "2023" OR Contains: "new" - Either current year or new items

Pro Tip: Combine with filename construction for powerful organization:
Example: Contains: "product-[A-Z0-9]{6}" to download only product SKUs matching a specific pattern

Important: Conditions are applied during download, not when pasting URLs. You'll still see all URLs in the list, but only matching ones will download.

11. Bulk Import via CSV

Powerful bulk task creation for managing large numbers of downloads efficiently.

Import Process

  1. Click "Choose File" and select your CSV
  2. CSV must include these columns (in any order):
    Column Description Example
    URL Image URL (one per line for multiple URLs) http://example.com/image.jpg or http://example.com/image1.jpg; http://example.com/image2.jpg for more than 1 images, separated by semi-colon
    File Types Semicolon-separated formats (or "all") jpg;png;webp
    Folder Name Target download folder Summer_Collection
    Download Time ISO 8601 format (YYYY-MM-DDTHH:MM) 2023-12-01T14:30
    ZIP Download TRUE/FALSE for zip packaging TRUE
    Build Filename TRUE/FALSE for custom filename construction TRUE
    Sortable List Filename components as HTML "filename;-;sequence"
    Sequence Start Initial number for sequencing 1 or 0050
    Download If Enabled TRUE/FALSE for conditional downloading TRUE
    Keywords Semicolon-separated conditions "contains: product;not-contains: thumb"
    Logic AND/OR for multiple conditions AND
    Status Initial task status pending
  3. Click "Import" to process the file
  4. The system will:
    • Validate license requirements (max 20 URLs in free version)
    • Create separate tasks for each CSV row
    • Preserve all settings exactly as specified

Exporting Templates

Recommended workflow:

  1. Create one task manually with your desired settings
  2. Use the Export function to generate a template CSV
  3. Duplicate and modify the template for bulk imports

Requirements:

  • CSV must use UTF-8 encoding
  • First row must be header row
  • When importing if you want multiple urls in a task make sure you separate them with a semicolon (;)
  • License restrictions apply (max URLs per task)

Advanced Tip: Use spreadsheet formulas to generate complex patterns:
Example: =CONCAT("product-",TEXT(ROW(A1),"000"),"-",TEXT(TODAY(),"yyyy")) creates sequential product filenames with current year
